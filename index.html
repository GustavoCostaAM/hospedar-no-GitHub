Passos para Hospedar um Projeto no GitHub Page

  1. Crie um repositório no GitHub com o mesmo nome do seu projeto.
  2. No terminal, inicialize o repositório local com $git init
  3. Altere o memso nome da branch principal para main
  4. Adicione os arquivos com git add . e faça o com git commit -m "first commit"
  5. Conecte ao repositório remoto com:
    git remote add origin https://github.com/seu_usuario/nome_do_repositorio.git
  6. Envie os arquivos com git branch -m origin main
